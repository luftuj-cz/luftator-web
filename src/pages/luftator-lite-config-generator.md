# LUFTaTOR Lite config generator

Použijte tento nástroj pro vygenerování konfiguračního souboru pro LUFTaTOR Lite.

<ConfigGenerator />

:::danger[Důležité]
Odpojte napájení předtím než připojíte LUFTaTOR Lite pomocí USB. Připojení LUFTaTORu USB kabelem současně se zapojeným napájením z adaptéru může poškodit Váš počítač i samotný LUFTaTOR Lite.
:::

- Pro nahrání konfigurace odpojte napájení a USB kabel od LUFTaTOR Lite.
- Stiskněte a držte tlačítko označené písmenem "B" na desce LUFTaTOR Lite.
- Poté připojte USB kabel a LUFTaTOR Lite se automaticky připojí k Vašemu počítači jako USB disk.
- Přetáhni vygenerovaný konfigurační soubor na tento disk.
- Po nahrání konfigurace se disk automaticky odpojí a LUFTaTOR Lite se restartuje.
